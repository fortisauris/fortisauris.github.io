Lekcia 3 -Zoznamy a Logika

  • Flexibilný zoznam - premenná LIST

    • Python3 dokáže pomocou premenných uchovať aj viacero hodnôt v zozname. Túto premennú nazývame LIST a môžeme ju flexibilne modifikovať a meniť. Vytvoríme si ju príkazom:

    zoznam_potravin = list()
    zoznam_potravin = []

    Ak máme vytvorený zoznam môžeme k nemu pridať na koniec nejaký obsah. Napríklad takto:

    zoznam_potravin.append("maslo") zoznam_potravin.append("maslo")

    alebo vložiť prvok na určené miesto ako do poličky v chladničke:

    zoznam_potravin.append("maslo") zoznam_potravin.insert(1, "jogurt") zoznam_potravin.insert(2, "maslo")

    Indexovanie LISTU

    Nezabudnite, že tak ako u STRINGU prvý člen LISTU má index 0. Ak chceme z LISTU vytiahnuť 3 člena musíme požiadať o člena s indexom 2.

    Chceme vedieť koľko máme v chladničke masla tak použijeme metódu triedy list count takto:

    zoznam_potravin.count("maslo")

    Python nám napíše, že v našej chladničke máme 2 maslá. To je správne. V LISTE sa môžu jednotlivé členy opakovať. Naša chladnička je nafukovacia, môže mať toľko členov koľko chce a nemusia to byť len potraviny ale aj iné premenné ako STRING, INTEGER, FLOAT, BOOLEAN, LIST. Áno náš zoznam môže ako člena obsahovať aj ďalší zoznam.
    Ak chceme z našeho zoznamu vytiahnuť jeho časť alebo len jedného člena použijeme podobne ako u STRINGU odkrajovanie SLICING:

    zoznam_potravin[1] zoznam_potravin[1:] zoznam_potravin[:1] zoznam_potravin[-1] zoznam_potravin[1,3]

    Člena LISTU vymažeme pomocou internej metódy remove takto:

    zoznam_potravin.remove("maslo")

    Pomocou tohoto príkazu sa nám odstráni jedno maslo zo zoznamu LIST. Na list vieme aplikovať aj príkazy ako len(), ktorý nám ukáže aktuálnu dĺžku našeho zoznamu.

  • Nemeniteľný zoznam - TUPLE

    • Niekedy je potrebné zadefinovať zoznam z ktorého bude program čerpať, ale nebude ho dynamicky meniť. Ušetrí sa čosi aj na pamäti, lebo TUPLE nezaberá v pamäti toľko miesta. Vytvoríme ho takto:

    konstanty = tuple()
    konstanty = ("maslo", "chlieb")

    Na vyberanie členov môžeme použiť SLICING a vyhľadávať členov podľa indexu:

    print(konstanty.index("chlieb"))

    Príkaz nám ukáže číslo na indexu na ktorom sa prvok nachádza. Ak by sme však hľadali neexistujúci prvok vyhodí chybu - ERROR

  • Premenná s neopakovanými členmi - SET

    • Python nám pomáha kde sa dá a pozná aj premennú kde nie je indexácia a zároveň sa jej prvky nemôžu opakovať. Vytvoríme ho:

    final_set = set()
    final_set = {1,2,3,4,5,6}

    Ak by sme do SETu zadali opakovane tú istú hodnotu jednoducho sa nevloží. Na SET nie je možné aplikovať ani SLICING. Je však možné do SETU pridať člena pomocou internej metódy add a postupne členov vyberať pomocou internej metódy pop. Napríklad:

    final_set.add("maslo")
    print(final_set.pop())

    Ak si dáme zobraziť final_set zistíme, že nám pribudol prvok ""maslo a prvok, ktorý sme metódou pop zo SETu vytiahli zmizol.
    Interná metóda pop nám umožňuje vyťahovať jednotlivé prvky ako to robí kúzelník z klobúka.
    Užitočnou internenou metódou SETU je možnosť porovnať dva SETY a vypísať spoločných členov. Spravíme to príkazom:

    second_set = {"maslo","chlieb","pomaranče"}
    final_set.intersection(second_set)

    Vytvorili sme si druhý set second_set a porovnali ho s final_set. Výsledkom by malo byť "maslo"

  • Počítač sa pýta - príkaz INPUT

    • V našej prvej lekcií sme pomocou príkazu print("Hello World!!!") zobrazili STRING. Ale Python umožňuje počítaču pýtať sa na niečo užívateľa programu. Používame na to príkaz input(). STRING, ktorý napíšeme do zátvorky sa užívateľovi zobrazí ako OTÁZKA alebo NÁVOD. Použijeme ho napríklad takto:

    input("Chceš zobraziť obsah chladničky? (A/N)")

    VÝSTUP príkazu input()

    Nezabudnite, že výstupom príkazu input() je vždy STRING. Pokiaľ potrebujete ako výstup číslo musíte výstup skonvertovať pomocou príkazu eval("3423").
    POZOR ak nebude STRING obsahovať iba číselné hodnoty s desatinnou čiarkou Python vyhodí chybu!

  • Jednoduchá logika pomocou IF, ELIF a ELSE

    • Iste si pamätáte ako sme spravili jednoduchý nekonečný cyklus pomocou while True: v minulej lekcií. Tento program sme ukončili kombináciou kláves Ctrl-C inak by bežal do nekonečna.
    Riešením je naprogramovanie jednoduchej logiky, ktorá nám ukončí prebeh nekonečného cyklu prostredníctvom príkazu break. Prvým prvkom logiky je podmienka ak - IF, ktorú použijeme nasledovne:

    a = False
    if a is True:
     # DO
     print(a)

    Kód v podmienke sa vykoná ak je splené, že premenná Boolean a je ň True. Ďalším prvkom našej logiky bude príkaz inak - ELSE, ktorý znamená, že ak podmienka IF nebola splnená vo všetkých ostatmných prípadoch vykonaj inštrukcie pod ELSE. Takto:

    a = True
    if a is True:
     print(a)
    else:  print("NEPRAVDA")

    Tretím členom našej logiky je príkaz ELIF . V zjednodušenom preklade ide o Inak ak a dopĺňa podmienku IF o ďalšie možnosti takto:

    a = True
    if a is True:
     print(a)
    elif a is False:  print("FALSE!!!") else:  print("NEPRAVDA")

    REŤAZENIE PODMIENOK input()

    if a is True or a is False:

     print(a)
