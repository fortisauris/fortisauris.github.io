Lekcia 3 -Zoznamy a Logika

  • Preferovaná premenná - Slovník DICTIONARY

    • Zoznamy, sety a tuples nám dávajú vcelku slušné možnosti ukladať si do pamäti počítača rôzne údaje a hodnoty. V Pythone však existuje ešte komplexnejšia premenná na ukladanie dát - Slovník. Tento nám umožňuje údajom priradiť kľúčové slovo a kedykoľvek ich podľa tohoto kľúčového slova nájsť a použiť. Vytvoríme si ho príkazom:

    chladnicka = dict()
    chladnicka["dvere"] = "maslo"

    V prvom riadku sme si vytvorili prázdny slovník a v druhom sme vlozili do slovníka prvú hodnotu s kľúčovým slovom dvere. Aby sme pochopili ako slovník funguje doplníme ďalšie údaje:

    chladnicka["policka1"] = ["slanina", "horcica", "slanina"]
    chladnicka["aktualna_teplota"] = 7

    Ako vidíme komplexnost Slovníka pozostáva v tom, že podobne ako u zoznamov, tuples a setov, môže byť členom slovníka akákoľvek premenná. Vypíšeme si jeho obsah na obrazovku pomocou príkazu print(chladicka)
    Na slovník môžeme aplikovať interné metódy napríklad:

    print(a.keys()) # interna metoda vypise vsetky kluce DICTIONARY
    print(a.get("aktualna_teplota")) # vytiahne hodnotu z dictu

    Hodnoty vieme aktualizovať napríklad aj spojením s iným slovníkom:

    b = dict()
    b["aktualna_teplota"] = 7.3
    a.update(b)
    print(a.pop("aktualna_teplota")) # vytiahne hodnotu z dictu
    print(a.values()) # vypíše hodnoty

    Ú L O H A

    Spravte v Pythone jednoduchý program, ktorý bude obsahovať jeden cyklus, dve ľubovoľné premenné a aspoň jednu logickú podmienku.

  • Definovanie funkcii pomocou def

    • Jednoduchý program sme zvládli, ale ako by sme v Pythone dokázali napísať niečo omnoho zložitejšie ? Jednoducho, Python nám totiž umožňuje definovať vlastné funkcie, ktoré sú vlastne podprogramy, ktoré si hlavný program dokáže kedykoľvek zavolať a použiť ich. Napríklad takto:

    chladnicka = {
      "policka1":["salama", "sunka", "slanina"],
      "policka2":["jogurt", "mlieko", "maslo"],
      "policka3":["mrkva", "zeler", "zemiaky"]
    }
    print(chladnicka)

    def vypis_policku(chladnicka: dict, cislo_policky: str):
      """
      Tento maly program sa vola funkcia a jeho ulohou je vypisat cislo cislo_policky
      z DICTIONARY chladnička.
      """
      nakupovane_potraviny = set() # definujeme prazdnu premennu SET
      for i in range(0, len(chladnicka[cislo_policky])): # Definujeme cyklus podla mnozstva v policke
       potravina = chladnicka[cislo_policky][i] # definujeme potravinu ako obsah
       print(potravina) # vypiseme ziskanu potravinu
       nakupovane_potraviny.add(potravina) # pridame do setu hodnotu potraviny
     print(nakupovane_potraviny)
     return nakupovane_potraviny
    print(vypis_policku(chladnicka, "policka1")) # zavolame a pouzijeme vytvorenu funkciu

    Kazdá funkcia je malý samostatný program uložený ako objekt v pamäti. Naša fukcia má dva vstupné údaje tzv. argumenty ARGS. Tieto použije vo svojom vnútri a pomocou príkazu vráť return vrátime výsledok do programu, ktorý funkciu zavolal. Nemusí to byť hlavný program. Môže to byť kľudne iná funkcia, ktorá potrebuje výpočet. Týmto spôsobom sa kód jazyka Python nielen sprehľadní pri spätnej analýze kódu - CODEREVIEW, ale pri ďalšom vývoji - DEVELOPMENTE , prepracovaní tzv. REFACTORINGU možno pridávať, zlepšovať, prepisovať kód funkcie bez zmien v hlavnom programe.

Gratulujeme! Dokážeme teraz všetky potrebné premenné programu držať v pamäti pomocou premennej slovník. A dokážeme programovať komplikovanejší program pomocou definovaných funkcií. Ešte pár krokov a budeme vedieť čítať kódy iných programátorov.
