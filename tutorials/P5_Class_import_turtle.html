  • Korytnačia grafika

    • Základných príkazov v Pythone nie je veľa. Predmetom tohoto krátkeho tutoriálu nie je prejsť ich všetky, ale naučiť sa spôsob ako fungujú a kde ich hľadať. Napriek tomu by sme zo sadou základných príkazov asi nevystačili. Preto sú tu takzvané MODULY, balíčky PACKAGES knižnice LIBRARY.
    Ich použitie a využitie v našom vlastnom kóde nám umožní pridať špecifické príkazy do Pythonu príkazom import. Napríklad ak chceme pridať k základným príkazom pythonu príkazy korytnačej grafiky dáme pythonu príkaz:

    import turtle

    K pythonu sa nám teraz pridali príkazy korytnačej grafiky, ktoré použijeme nasledovne:

    import turtle

    Teraz môžeme napísať jednoduchý program s použitím príkazov korytnačej grafiky:

  • Triedy a ich metódy

    • Pri písaní našeho programu časom narazíme na to, že postupným pridávaním funkcí sa stáva program neprehľadný. Funkcie nám tiež umožňujú využívať premenné, ale samy uchovávať data nevedia. Aby v tom bol poriadok existujú v Pythone triedy CLASS. Môžu obsahovať funkcie ako časti programu ale aj samotné dáta, ktoré môžu meniť a poskytovať iným častiam programu.
    Triedu preto možno zjednodušene opísať ako plánik, alebo náčrtok. Aby sa tento plán stal skutočnosťou treba ho iniciovať a vytvoriť jeho tzv. inštanciu INSTANCE. Napíšeme si jednoduchú dátovú triedu aj so základnými metódami: 

    class NasaTrieda(object):
    """
    Dátová trieda obsahujúca dáta o našej triede na ZS
    """

    def __init__(self):
        """
        Magicka metoda __init__ nastavuje kod, ktory sa ma vykonat pri iniciacii
        Triedy. Definuje aj premenne triedy.
        """
        self.menny_zoznam_ziakov = list()  # definuj premennu triedy ako prazdny list
        self.predmety = set()  # definuj vyucovacie predmety ako prazdny set.

    def pocet_ziakov(self):
        """
        Metoda triedy NasaTrieda pouzije menny zoznam na vypocet poctu ziakov v zozname

        """
        return len(self.menny_zoznam_ziakov)

    def predmety_getter(self):
        """
        Metoda triedy NasaTrieda ziskava (get) informacie z premennej triedy a
        prezentuje ich v pozadovanom tvare.
        param :: none
        return:: none
        """
        for i in self.predmety:  # cyklus prechadza vsetkych clenov self.predmety
            print(i)  # tlaci jednotlive predmety do riadkov pod seba
        return  # vracia sa do hlavneho programu. ZIADNA HODNOTA

    def predmety_setter(self, value: str):
        """
        Metoda triedy NasaTrieda umoznuje pridat value do setu self.predmety
        parameter:: value - string nazov predmetu
        return:: none
        """
        self.predmety.add(str(value))  # prida string value do premennej self.predmety

    def predmety_deleter(self, value: str):
        """
        Metoda triedy NasaTrieda umoznuje odstranit zaznam z premennej self.predmety
        parameter:: value - string nazov predmetu, ktory sa vymaze
        return:: none
        """
        while value in self.predmety:  # zacni cykls ak sa value nachadza v self.predmety
            self.predmety.remove(value)  # odstran hodnotu z self.predmety
            return  # vracia sa do hlavneho programu ZIADNA HODNOTA

    def je_ziak_clenom_triedy(self, meno_ziaka: str):
        """
        Metoda zistuje ci je ziak clenom Triedy
        parameter:: meno_ziaka STRING
        return:: Odpoved Pravda alebo Nepravda  BOOL
        """
        result = meno_ziaka in self.menny_zoznam_ziakov  # nachadza sa meno ziaka v mennom zozname triedy ?
        # print(result)
        return result  # vrati hodnotu True alebo False do hlavneho programu

    Metódy, ktoré nevyužívajú premenné triedy CLASS VARIABLES sa delia na statické STATICMETHOD a triedne CLASSMETHOD. Oba tieto druhy metód môžeme využiť aj bez iniciacie triedy. To nám umožňuje napísať triedy, ktoré budú plné užitočných nástrojov ana pomocné operácie a prístupné pre ktorúkoľvek časť programu. Aha takto: 


    class FileTools(object):

    @staticmethod
    def file_load(filename_w_path: str):
        """
        Metoda triedy FileTools otvori subor na citanie v mode READ a kodovanim
        textu utf8. Pomocou prikazu read() nacita cely jeho obsah a ulozi ho
        do premennej result. Na konci metody tuto premennu vrati do hlavneho
        programu.
        parameter ::  filename_w_path STRING meno suboru aj s cestou k nemu
        return :: result STRING obsah nacitaneho suboru
        """
        result = str()
        f = open(file=filename_w_path, mode='r', encoding="utf8")
        result = f.read()
        print("SUBOR SA NACITAL")
        return result

    @classmethod
    def file_save(cls, filename_w_path, plaintext):
        """
        Metoda otvori subor na pisanie v mode WRITE a kodovanim textu v utf8.
        Pomocou prikazu write ulozi text dodany v STRINGU plaintext. Na konci
        vrati hodnotu True, ktora pre hlavny program znamena, ze sa subor uspesne
        ulozil.
        """
        f = open(file=filename_w_path, mode='w', encoding="utf8")
        f.write(plaintext)
        print("SUBOR SA ULOZIL")
        return True

# Vyuzijeme staticku a triednu metodu
FileTools.file_save("test.txt", "Hello from Program!!!")
nacitany_text = FileTools.file_load("test.txt")
print(nacitany_text)


    Všimnite si, ze zmizlo self u STATICKEJ metódy nakoľko sa neodvoláva na svoju inštanciu a u TRIEDOVEJ metódy sa objavilo cls v zátvorke to znamená, že sa odvoláva nie na ištanciu triedy ale iba na jej plánik CLASS.
    V programe sme si predstavili príkaz open, ktorý má na starosti vstupy a výstupy do súborov. Módy rREAD a wWRITE znamenajú, že bude zapisovať text. Ak by sme chceli zapisovať binárne dáta (jednotky a nuly)museli by sme dáta najprv previest do bytes() a uložiť v móde wb WRITE BINARY alebo rb READ BINARY.

    ČUDNÉ OZNAČENIE @staticmethod a @classmethod

    Iste ste si všimli divné označenie metód v triede FileTools. Ide o tzv. dekorátori - DECORATORS. Sú to funkcie, ktoré dokážu ako argument prijať ďalšiu funkciu a modifikovať jej priebeh. Zatiaľ Vám stačí vedieť, že existujú no čoskoro už napíšete vlastný dekorátor v Pythone.
